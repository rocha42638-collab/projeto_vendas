Análise de Performance de Vendas (Data Science)
Descrição do Projeto
Este projeto consiste em uma análise exploratória de dados (EDA) focada em performance comercial. Utilizando Python, transformei uma base de dados bruta em um dashboard de insights estratégicos para suporte à tomada de decisão.

O foco principal foi identificar padrões de faturamento e, principalmente, a consistência de vendas entre os membros da equipe, utilizando métricas estatísticas de dispersão.

Insights Extraídos
1. Ranking de Faturamento
Identificamos os líderes de venda em volume financeiro absoluto. Este gráfico permite uma visão rápida de quem está trazendo mais receita para a operação.

2. Análise de Consistência (Boxplot)
Diferente do ranking simples, o Boxplot nos permite visualizar a estabilidade do vendedor.

Insight: Vendedores com caixas menores e menos "pontos fora da curva" (outliers) possuem uma estratégia de preço e vendas mais previsível e segura para a empresa.

3. Mix de Produtos (colaborador Anderson Gomes)
Uma análise profunda sobre o perfil do vendedor Anderson Gomes, identificando em quais nichos de produtos ele possui maior conversão.
