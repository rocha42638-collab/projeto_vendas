 Análise de Desempenho de Vendas (Data Science)
 Descrição do Projeto

Este projeto consiste em uma Análise Exploratória de Dados (EDA) focada em desempenho comercial ao longo de 6 anos de vendas. Utilizando Python, a base de dados bruta foi transformada em insights estratégicos e um dashboard interativo para suporte à tomada de decisão.

O projeto tem como foco principal identificar padrões de faturamento, sazonalidade de vendas, desempenho de vendedores e dependência de produtos no faturamento da empresa.

 Principais Insights de Negócio
Ranking de Faturamento: Identificação dos vendedores com maior geração de receita.
Consistência de Vendas (Boxplot): Identificação de vendedores com maior estabilidade de preços e menor risco comercial.
Sazonalidade: O mês de março apresenta o maior faturamento, enquanto fevereiro apresenta o menor, com crescimento médio de aproximadamente 21% de fevereiro para março.
Produto Carro-Chefe: O Produto D representa aproximadamente 20% do faturamento no mês de março, sendo o principal gerador de receita.
Mix de Produtos: Identificação de dependência de produtos específicos no faturamento total.
Forma de Pagamento: Análise do impacto financeiro das modalidades de pagamento, com oportunidade de recuperação de margem via PIX.

Tecnologias e Ferramentas
